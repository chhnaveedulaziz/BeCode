## BE Payments API
    Implementation of payment API
### Pre-requisites

- Node js ~ v16.0 
- Postgres
- Yarn

### Installation

Install the dependencies and devDependencies and start the server.

- run `yarn`
- copy `env.example` to `.env`

To start server: 
    - run `yarn start`

To use API:
     - Hit relevant endpoint `http://localhost:4000/<endpoint>`
